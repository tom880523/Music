自訂聲音節奏播放器

這不是固定鼓組。左邊每一列都是你自行新增的「音源」，例如杯子撞擊、敲桌子、拍手、玻璃聲、環境音。點擊格子設定何時播放該音源。

音源管理

新增音源後，所有播放組都會出現同一個音源列，但每個播放組可以有不同節奏格子設定。

YouTube 段落編排

可設定某個時間段使用哪一個播放組，例如 0:00–1:00 用 A 組，1:00–2:00 用 B 組。播放 YouTube 時，系統會依照目前時間自動切換播放組。

JSON 目前會保存已上傳音檔，方式是把音檔轉成 Data URL 寫進 JSON。好處是單一 JSON 就能恢復專案；缺點是音檔越多，JSON 會越大。
